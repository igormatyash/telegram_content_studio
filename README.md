# VoicerHub Content Bot

Редакційний Telegram-бот, який створює українські пости та брендовані
зображення для каналу `@voicerhub`. Публікація відбувається тільки після
натискання кнопки редактором.

## Що вміє MVP

- генерує пости окремо для Voicer, TONY або загальних AI/CX тем;
- використовує затверджену базу фактів і не повинен вигадувати функції;
- створює AI-візуал без тексту, а потім накладає точний заголовок і бренд;
- показує приватне прев'ю з кнопками `Опублікувати` та `Інша версія`;
- відправляє тексти й зображення через OpenAI Batch API;
- дозволяє окремо обирати модель для тексту та зображення;
- підтримує бібліотеку логотипів і візуальних референсів для GPT Image 2;
- показує batch-статуси, чергу та витрати у веб-адмінці;
- зберігає історію заголовків у SQLite, щоб зменшити повтори;
- допускає до редактора тільки Telegram ID з `ADMIN_USER_IDS`.

## Безпечне налаштування

1. У BotFather виконайте `/revoke` для токена, який потрапив у чат.
2. Створіть новий токен.
3. Додайте нового бота адміністратором `@voicerhub` лише з правом публікації.
4. Скопіюйте `.env.example` у `.env` і заповніть секрети локально.
5. Спочатку залиште `ADMIN_USER_IDS` порожнім, запустіть бота та надішліть `/id`.
6. Зупиніть бота, внесіть отриманий ID у `.env` і запустіть знову.

Не надсилайте новий Telegram-токен або OpenAI API key у чат.

## Запуск

```bash
cd voicerhub-content-bot
python3 -m venv .venv
.venv/bin/pip install -e ".[dev]"
cp .env.example .env
# Заповніть .env
.venv/bin/voicerhub-bot
```

## Команди

```text
/id
/draft tony Як голос допомагає побачити причину відмови
/draft voicer Як перетворити відгуки на рішення
/draft general Практичне використання синтетичних даних
/jobs
/preview 12
```

## Адмінка

Локально:

```bash
.venv/bin/voicerhub-admin
```

Після серверного деплою: `http://SERVER_IP/content-admin/`.
Доступ захищений значеннями `ADMIN_USERNAME` і `ADMIN_PASSWORD` з `.env`.
Ці значення створюють лише першого адміністратора в порожній базі. Далі
користувачі, ролі, блокування та зміна паролів керуються в розділі
`Користувачі`. Паролі зберігаються як salted `scrypt` hashes, а вхід працює
через HTTP-only cookie-сесію.

Після підключення HTTPS встановіть `SESSION_COOKIE_SECURE=true`.

Вартість є розрахунковою: для тексту вона базується на фактичних input/output
tokens із Batch API, для зображення — на заданій ціні одного рендера. Остаточні
витрати звіряйте з OpenAI Billing.

Зображення без референсів генеруються через Batch API. Якщо вибрано логотипи
або приклади, система використовує Image Edits із високою точністю входів.

## Docker

```bash
docker compose up -d --build
```

Compose підключається до наявної мережі `worldcup-2026_default`, щоб існуючий
Caddy міг проксувати `/content-admin/` на контейнер `voicerhub-admin`.

## Важливе обмеження Telegram

Підпис до фото має жорсткий ліміт, тому генератор навмисно створює компактні
пости до 950 символів. Це дозволяє картинці й тексту залишатися одним дописом,
а не двома окремими повідомленнями.

## Наступний етап

Після перевірки якості MVP можна додати календар публікацій, автоматичний
пошук новин із перевіркою джерел, веб-редактор і кілька фірмових візуальних
шаблонів.
